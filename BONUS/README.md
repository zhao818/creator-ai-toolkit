## 1. 发布前检查清单（要点预览）

每次推送前 60 秒逐项勾过：

### 内容层
- 标题 ≤ 9 中文字
- 摘要 ≤ 17 中文字
- 核心命题句独立可站住
- 开头 3 秒钩子到位
- 至少 1 句可传播金句
- 结尾回扣开头
- 无 AI 套话

### 排版层
- ① 开篇诗行 ← 蓝底 + 白字 + 暖金收尾
- ② 金句 × 3 ← 蓝底 + 金左边框 + ✦
- ③ 分区标题 ← 壹贰叁 + 蓝底 + 白字
- ④ 付费钩子（可选）← 红虚线框
- ⑤ 收尾诗行 ← 蓝底 + 暖金
- ⑥ 品牌尾签 ← ✦ + 美好需要创造 + 世界一隅 + 日期

### 传播层
- 转发钩子至少 3 处
- 收藏引导在结尾前
- 评论提问在结尾
