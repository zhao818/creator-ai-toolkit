# 工作流 ③ · 内容融合术

> 借壳生蛋，以我为主
>
> 别人的热点（壳） + 你的原创思想（核） = 既好传播又有深度的内容

---

## 核心理念

```
❌ 只写自己的东西 → 选题枯竭、没人搜、自嗨
❌ 只转别人的东西 → 没有品牌辨识度、为他人做嫁衣
✅ 借用热点话题做壳 → 抛出自己的原创思想 → 读者奔着热点来，带着你的思想走
```

---

## 融合公式

```
[热门话题/人物/事件] + [独特解读角度] + [你的原创观点] = 融合文章

三个层次：
├── 第一层（壳）：大家都知道的事 —— 吸引点击
├── 第二层（桥）：独特的解读角度 —— 制造反差
└── 第三层（核）：你的原创思想 —— 这才是你的东西
```

---

## 四种融合模式

### 模式 A：热点事件 → 哲学升华

```
壳：一个大家都在讨论的热点事件
桥：从事件中提炼出一个普遍的困境/矛盾
核：用你的思想体系给出独特的解读
```

**实例**：
| 层次 | 内容 |
|------|------|
| 壳 | "DeepSeek 爆火，程序员担心失业" |
| 桥 | "但我们真正焦虑的不是被替代，而是不知道自己除了写代码还能做什么" |
| 核 | "✦ 你的价值从来不在你的技能里——在你的眼光里" |

**结构**：
```
① 开篇：描述热点事件（~300字，让人知道你在说什么）
② 转弯：从事件提炼普遍困境（~200字，"你不是一个人"）
③ 展开：2-3 层递进解读（每层配金句）
④ 收尾：回到热点，但读者已经有了新的眼睛
```

### 模式 B：技术工具 → 人文反思

```
壳：一个新工具/技术/产品
桥：这个工具解决了什么问题？但创造了什么新问题？
核：效率之外，我们真正需要的是什么？
```

**实例**：
| 层次 | 内容 |
|------|------|
| 壳 | "又一个一键生成 PPT 的 AI 工具" |
| 桥 | "它让你 3 分钟做完 PPT，但你花 3 小时纠结为什么自己的 idea 这么薄" |
| 核 | "✦ 工具省下的时间，如果不用来深度思考，只会让浅薄加速繁殖" |

### 模式 C：思想人物 → 技术印证

```
壳：一个思想家/作家/历史人物的故事
桥：ta 的困境/洞察，在今天有了新的版本
核：古老的智慧依然生效——但需要新的表达
```

**实例**：
| 层次 | 内容 |
|------|------|
| 壳 | "庄子说'吾生也有涯，而知也无涯'" |
| 桥 | "2000 年后，我们用 AI 来应对知识的无限——但焦虑没变" |
| 核 | "✦ 以有涯随无涯，殆矣——AI 时代庄子的警告更响了" |

### 模式 D：跨界碰撞 → 新视角

```
壳：两个看似毫无关联的事物
桥：在某个维度上，它们是同一件事
核：这个维度，就是你的核心思想
```

**实例**：
| 层次 | 内容 |
|------|------|
| 壳 | "CAD 画线 + 写文章" |
| 桥 | "都是先搭框架，再填细节，最后反复改" |
| 核 | "✦ 所有的创造，底层都是同一件事：把模糊变成精确" |

## 素材收集系统

### 素材源

**技术/热点类**：
| 来源 | 收集什么 | 怎么用 |
|------|---------|--------|
| GitHub Trending | 本周最火的开源项目 | "又一个 AI 工具爆了——但真正危险的不是工具本身" |
| Hacker News | 技术圈讨论热点 | 切入技术人的焦虑/困境 |
| V2EX / 掘金热榜 | 中文技术圈在讨论什么 | 中文社区的情绪和争议 |
| Product Hunt | 新产品/新工具 | "这个产品 3 天 10 万用户——但底层逻辑 500 年前就有了" |
| 知乎热榜 | 大众在焦虑/争论什么 | 找到技术和生活的交叉点 |
| 即刻/微博热搜 | 中文互联网情绪 | 捕捉当下的集体情绪 |

**思想/哲学类**：
| 来源 | 收集什么 | 怎么用 |
|------|---------|--------|
| Aeon / Psyche | 深度思想文章 | 用技术视角解构哲学问题 |
| The Marginalian | 文学+科学+哲学交叉 | 直接借鉴融合方式 |
| 经典重读 | 老子/庄子/尼采/海德格尔 | "2000 年前老子说的一句话，解释了今天的 AI 焦虑" |
| 人物传记/纪录片 | 有趣的人物故事 | 人物弧线 + 思想提炼 |

### 收集节奏

```
每天（10 分钟）：
├── 刷一遍 GitHub Trending / 知乎热榜 / 即刻
├── 看到有意思的 → 丢进选题池（只记标题 + 一句话为什么有意思）
└── 不深入读，只做标记

每周（30 分钟）：
├── 从选题池中挑 3-5 个最有潜力的
├── 深入读素材原文
└── 判断：能不能和你的思想体系产生化学反应？

每月（1 小时）：
├── 整理本月素材 → 发现重复出现的主题
├── 重复出现的主题 = 社会正在关心的 = 你应该写的
└── 更新融合选题池
```

## 融合文章写作模板

### 标题公式（≤9 字）

| 模式 | 公式 | 示例 |
|------|------|------|
| A | 「[人物/事件]的[反常识解读]」 | 「她的52部错爱」「AI的庄子解法」 |
| B | 「[工具]的另一面」 | 「效率的另一面」「AI不问的事」 |
| C | 「[古人]没说的事」 | 「庄子没说的事」「老子懂AI」 |
| D | 「[A]与[B]之间」 | 「画线与写作」「代码与孤独」 |

### 正文结构（融合版 SCQA）

```
S (壳)   情境：大家都看到了什么 —— 热点/事件/工具/人物
         "全网都在讨论XX..."

C (桥)   冲突：但你有没有想过另一个角度 —— 制造反差
         "但所有人都在讨论功能，没人问一个问题：..."

Q (转折) 问题：这个问题指向什么 —— 引出你的思想体系
         "这其实不是在问XX，而是在问：..."

A (核)   答案：你的原创思想 —— 这就是你的东西
         分 2-3 层展开，每层配金句
         "✦ [金句]"

收尾     回到壳，闭环 —— 读者发现自己已经不一样了
         "下次你再看到XX的时候，你看到的就不再是XX了"
```

## AI Prompt · 融合选题生成

```markdown
## Prompt 模板

我是一位写作者，我的核心思想体系围绕以下概念：
- [你的核心概念 1]：[一句话解释]
- [你的核心概念 2]：[一句话解释]
- [你的核心概念 3]：[一句话解释]

最近我注意到以下热点话题：
1. [热点 1]
2. [热点 2]
3. [热点 3]

请帮我为每个热点找到 2-3 个融合角度：
- 这个热点能和我的哪个核心概念产生化学反应？
- 连接角度是什么？（桥）
- 最终能提炼出什么原创观点？（核）
- 这个角度够意外吗？（读者能猜到吗？）

输出格式：
热点：[热点名称]
├── 角度 1
│   ├── 壳：[一句话]
│   ├── 桥：[一句话]
│   ├── 核：[一句话]
│   └── 标题建议：[≤9字]
├── 角度 2
│   └── ...
```

## 融合自检清单

写融合文章前逐条勾过：

- [ ] **壳够热吗？** — 目标读者最近在讨论/搜索这个话题吗？
- [ ] **桥够意外吗？** — 连接壳和核的角度让人"没想到"吗？（意料之外 + 情理之中 = 最佳）
- [ ] **核是原创的吗？** — 最终表达的是你自己的思想，不是复述别人
- [ ] **有金句吗？** — 至少 1 句能独立传播
- [ ] **壳核闭环了吗？** — 开头提的话题，结尾有没有给出新的看待方式？
- [ ] **不依赖壳成立吗？** — 把热点拿掉，文章本身还有没有价值？（如果没有 = 纯蹭热度，不写）
- [ ] **有品牌印记吗？** — 你的审美、调性、态度是否在场？

## 即兴示范

### 用模式 A 写一篇

**素材**：
- 热点：DeepSeek V4 发布，全网讨论 AI 又变强了
- 哲学：庄子「吾生也有涯，而知也无涯」
- 我的思想：对效率和深度的独特理解

**融合结果**：

> **标题**：AI的庄子解法
>
> **开篇（壳）**：
> DeepSeek V4 发布了。朋友圈又是一片"AI要取代人类了"。
>
> **转弯（桥）**：
> 但你有没有发现——工具越来越强，人的焦虑也越来越大。AI 能一秒读完一百本书，但你多久没完整读完一本了？
>
> **展开（核）**：
> 2000 多年前，庄子说了一句话：「吾生也有涯，而知也无涯。以有涯随无涯，殆矣。」
>
> 用有限的生命去追逐无限的知识，这是危险的。
>
> AI 解决的是「无涯」的问题——它能读尽天下书。但它解决不了「有涯」的问题——你的注意力、你的时间、你的深度思考能力，依然有限。
>
> 真正危险的，从来不是 AI 越来越强。而是我们把自己的「有涯」，也拱手让了出去。
>
> **收尾**：
> 下次你打开 AI 工具的时候，问自己第一个问题不是「它能帮我做什么」——而是「我为什么要做这件事」。
>
> ✦ 以有涯随无涯，从来都是危险的。AI 时代，庄子的警告比 2000 年前更响。
