## 真实案例：改之前 vs 改之后

### ❌ 改之前 · 《生育焦虑的背后》（草稿）

> 当社会在为出生率下降而焦虑时，这场对话早已超越了人口统计学。讨论的起点看似简单，却层层递进。第一层是人口结构、养老负担。第二层是社会自救、技术替代。第三层是人活着究竟为了什么……

**问题**：无钩子、无章节分区、无金句、术语不统一。

### ✅ 改之后 · 《生育焦虑的第三层》

> （钩子）你焦虑的不是生不生——是生了之后，你是谁。
>
> **壹 · 第一层**：人口数字在说话。养老金缺口、劳动力萎缩——这些是你看得到的焦虑。但它们只是表层。
>
> **贰 · 第二层**：社会在自救。鼓励生育的政策一波接一波。但这一层仍然在问"怎么让人多生"——而不是"为什么人不愿意生"。
>
> **叁 · 第三层**：我32岁那年，坐在阳台上盯着对面的楼看了两个小时。不是在想生不生的问题。是在想：如果我所有的身份都被拿走，我还剩什么？
>
> ✦ 真正的生育率问题，不在子宫里——在意义里。
>
> 当一个人连自己的存在意义都找不到的时候，你让她怎么敢把另一个生命带到这个世界？
