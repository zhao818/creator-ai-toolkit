## 工作流 ① · AI 写作全流程

### 从「灵光一闪」到「可发布文章」，7 步走完

```
灵感碎片 → 选题判断 → 结构搭建 → AI 辅助扩写
   → 人工深加工 → 品牌排版 → 发布前自检
```

### 核心 Prompt 模板（节选）

**步骤 1：选题判断**
```
我有一个写作灵感：「____」。请帮我：
1. 判断这个选题的读者覆盖面（窄/中/宽）
2. 找出 3 个可能的独特角度
3. 这个选题最容易写成哪种类型（教程/观点/故事/分析）
```

**步骤 3：结构搭建**
```
基于选题「____」和角度「____」，请帮我搭建文章结构：
- 用 SCQA 框架（情境→冲突→问题→答案）
- 每个部分标注预估字数
- 标注需要插入金句的位置（至少 3 处）
```

**步骤 6：品牌排版**
```
将以下 Markdown 内容转换为我指定的品牌 HTML 格式。
品牌色：深海蓝 #1a1a2e + 暖金 #c8a03c
要求：6 区块结构（开篇诗行 → 金句 → 分区标题 → 正文 → 收尾诗行 → 品牌尾签）
```

### 发布前自检清单
- [ ] 核心命题句：能独立站住的一句话 thesis
- [ ] 开头有钩子（3 秒让人停住）
- [ ] 每 300 字有路标（小标题/金句/转折）
- [ ] 至少有 1 句可独立传播的金句
- [ ] 结尾回扣开头形成闭环
- [ ] 排版 6 区块完整
- [ ] 标题 ≤ 9 字，摘要 ≤ 17 字

### 真实案例

**改之前（草稿）**：
> "当社会在为出生率下降而焦虑时，这场对话早已超越了人口统计学..."

**改之后（深加工）**：
> 壹 · 生育焦虑从来不只是关于生不生
> 贰 · 三个层次：人口→社会自救→人为什么活着
> 叁 · 结语：能够这样思考，本身就证明人不只是繁衍的载体
> ✦ 真正的生育率问题，不在子宫里——在意义里。

## 工作流 ② · 品牌排版一键生成

### 深海蓝 + 暖金 · 6 区块排版系统

你的文章写完 Markdown，剩下的交给这套模板。6 个区块按顺序组装，每篇输出风格统一、品牌辨识度拉满。

### 区块结构

```
① 开篇诗行    ← 深海蓝全幅 + 白字分行 + 暖金粗体收尾
② 金句 × 3    ← 深海蓝全幅 + 金左边框 + ✦ 前缀
③ 分区标题    ← 深海蓝全幅 + 金左边框 + 白字壹贰叁编号
④ 正文内容    ← 15px/2.0行高/首行缩进
⑤ 收尾诗行    ← 深海蓝全幅 + 暖金分行
⑥ 品牌尾签    ← ✦ + 美好需要创造 + 世界一隅 + 日期
```

### CSS 代码（复制即用）

**开篇诗行**
```html
<section style="background:#1a1a2e;padding:32px 20px 28px 20px;margin-bottom:24px;">
<p style="color:#c8a03c;font-size:12px;letter-spacing:6px;text-align:center;">世 界 一 隅</p>
<p style="color:#fff;font-size:16px;line-height:2.2;text-align:center;">
分行诗行<br>
<span style="color:#c8a03c;font-weight:bold;">收尾金句</span>
</p>
</section>
```

**品牌尾签**
```html
<p style="text-align:center;margin:36px 0 12px;">
<span style="color:#c8a03c;font-size:16px;">✦</span>
</p>
<p style="text-align:center;font-weight:700;color:#1a1a2e;">美好需要创造</p>
<p style="text-align:center;color:#c8a03c;font-size:12px;">世界一隅 · WORLD CORNER</p>
```

### 关键约束
- 标题 ≤ 9 中文字
- 摘要 ≤ 17 中文字
- 公众号编辑器用**内联样式**（不支持 class）
- 传 API 时必须 `ensure_ascii=False` + `charset=utf-8`（否则中文乱码）

## 工作流 ③ · 内容融合术

### 解决"不知道写什么"——借壳生蛋，以我为主

```
别人的热点（壳） + 你的原创思想（核） = 既好传播又有深度的文章

❌ 只写自己的 → 选题枯竭、没人搜、自嗨
❌ 只转别人的 → 没有品牌辨识度
✅ 借热点做壳 → 抛出原创思想 → 读者奔热点来，带着你的思想走
```

### 四种融合模式

| 模式 | 公式 | 示例 |
|------|------|------|
| **A. 热点→哲学** | 热点事件 + 普遍困境 + 你的解读 | "DeepSeek 爆火 → 我们焦虑的不是被替代，而是不知道自己还能做什么" |
| **B. 工具→人文** | 新工具 + 它解决了什么 + 创造了什么新问题 | "一键生成 PPT → 3分钟做完，但花 3 小时纠结为什么 idea 这么薄" |
| **C. 古人→今用** | 经典智慧 + 今天的版本 + 你的视角 | "庄子说以有涯随无涯，殆矣 → AI 时代这个警告更响了" |
| **D. 跨界碰撞** | A + B + 共通底层逻辑 | "CAD画线 + 写文章 → 都是先搭框架再填细节最后反复改" |

### 融合标题公式（≤9 字）
- 「[人物]的[反常识]」 → 「她的52部错爱」
- 「[工具]的另一面」 → 「效率的另一面」
- 「[古人]没说的事」 → 「庄子没说的事」

### 素材收集节奏
```
每天 10 分钟：刷 GitHub Trending / 知乎热榜 / 即刻 → 丢选题池
每周 30 分钟：挑 3-5 个，深入读 → 判断能否和你的思想体系产生反应
每月 1 小时：发现重复出现的主题 = 社会正在关心的 = 你该写的
```

### 融合自检
- [ ] 壳够热吗？读者最近在讨论这个话题吗？
- [ ] 桥够意外吗？连接角度让人"没想到"吗？
- [ ] 核是原创的吗？表达的是你自己的思想？
- [ ] 有金句吗？至少 1 句能独立传播？
- [ ] 不依赖壳，文章本身还有价值吗？
